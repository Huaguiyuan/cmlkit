*****
Grids
*****

Imports
=======

``cmlkit.autotune.grid``, or using ``gr_`` in an ``autotune`` config.

Introduction
============

This module implements some common (log)grids as shortcuts to ``np.logspace``. This will mostly be used by ``autotune``, where the functions listed below have to be prefixed with ``gr_``.
