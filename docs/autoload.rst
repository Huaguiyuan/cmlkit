********
Autoload
********

Imports
=======

``cmlkit.autoload``, but ``cmlkit.load_dataset`` is provided for convenience

Introduction
============

Autoload exists to automatically load datasets from a central location specified by the ``$CML_DATASET_PATH`` environment variable (see :doc:`globals`) or the current working directory. At a later date, this might be extended to models.
